
 🎬 Movie Revenue Analysis using Python (Pandas EDA)

## 📖 Overview

This project performs Exploratory Data Analysis (EDA) on a movies dataset containing information about budget, gross revenue, ratings, and release year.

The objective was to clean the dataset, analyze relationships between financial and rating metrics, and derive insights into factors affecting box office success.

---

## 🛠 Tools Used

- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  

---

## 🧹 Data Cleaning Steps

- Removed duplicate records  
- Handled missing values  
- Converted incorrect data types (object → numeric)  
- Standardized column names  
- Checked for outliers  

---

## 📊 Analysis Performed

- Pearson Correlation Analysis  
- Budget vs Gross Revenue Analysis  
- Ratings vs Revenue Comparison  
- Year-wise Revenue Trends  

---

## 🔍 Key Insights

- Higher budget movies tend to generate higher gross revenue.  
- Ratings show moderate correlation with revenue.  
- Certain years recorded significantly higher box office collections.  

---

## 🚀 Conclusion

This project demonstrates how data cleaning and correlation analysis can uncover meaningful insights from real-world datasets using Python.
